Exercícios Aula 01: AI Task System (ClickUp-like)

Construção de um mini sistema de gestão de tarefas com inteligência artificial inspirado em ClickUp. O objetivo é transformar linguagem natural em tarefas estruturadas e automáticas.

PARTE 1: AI para criação e melhoria de tarefas

Exercício 1 — Criador de Tarefas Inteligente

Cria a função:

createTaskFromText(text)

O objetivo é transformar texto livre em uma tarefa estruturada, como num sistema real de gestão de trabalho.

Input:

Preciso de corrigir o bug do login que está a falhar para vários utilizadores e é urgente

Output esperado:

{
  "title": "Corrigir bug no login",
  "description": "Erro no login afeta vários utilizadores",
  "priority": "high",
  "tags": ["bug", "login"]
}
Este é o núcleo de qualquer sistema AI de produtividade: transformar linguagem humana em dados estruturados.

Exercício 2 — Melhoria Inteligente de Tarefas

Cria a função:

refineTask(task)

A AI deve melhorar tarefas já existentes, tornando-as mais claras, completas e profissionais.

Input:

{
  "title": "Bug no login",
  "description": "login não funciona",
  "priority": "high",
  "tags": ["bug"]
}

Output esperado:

{
  "title": "Corrigir falha no sistema de login",
  "description": "Utilizadores estão impossibilitados de fazer login devido a erro no sistema de autenticação. Deve ser investigado e corrigido o fluxo de login.",
  "priority": "high",
  "tags": ["bug", "authentication", "login"]
}
Simula funcionalidades reais de ferramentas como ClickUp, Notion ou Jira AI: melhorar tarefas automaticamente.

PARTE 2: API (Mini ClickUp Backend)

Exercício 3 — Criar Tarefa via API

POST /api/tasks/create
Objetivo: Receber texto livre e criar automaticamente uma tarefa estruturada com AI. Simula criação de tarefas no ClickUp.
POST /api/tasks/refine
Objetivo: Melhorar tarefas existentes usando AI, tornando-as mais claras e completas antes de serem executadas pela equipa.
POST /api/tasks/summarize
Objetivo: Resumir descrições longas de tarefas em uma frase simples e objetiva para facilitar leitura rápida.
POST /api/tasks/suggest-tags
Objetivo: Analisar uma tarefa e sugerir automaticamente tags relevantes para melhor organização.

Critérios de sucesso

  • Uso de try/catch nas chamadas à API
  • Validação de input antes de enviar para AI
  • Uso de variáveis de ambiente (.env)
  • Endpoints testados com Postman
  • Respostas estruturadas em JSON consistente
